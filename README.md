 # スマートお迎え
 ２年前期制作のAndroidアプリ<br>
 ３人でのグループ開発


## 成果

**信州未来アプリコンテスト　最終審査へ進出**

コンテストサイト：https://shinshu-futureapp.net/information/contest/

## 概要
保護者が保育園に近づいたら、音声でお知らせするアプリ<br>
保育園に設置されたWi-Fiを保護者のスマートフォンで検知し、Wi-Fi経由で保育園へ通知。音声で園児の呼び出しを行う<br>
